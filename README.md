## Installation

1. Install the [Tmux Plugin Manager (TPM)](https://github.com/tmux-plugins/tpm)
2. Add the following to your *.tmux.conf* file:

    ```
    set -g @plugin 'openjck/tmux-townsend'
    ```
3. Press `prefix + I`
